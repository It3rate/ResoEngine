# Core 2 Repetition

This folder contains the continuation and repetition substrate of Core 2.

Repetition here is broader than exponentiation. It includes any lawful “again”:

- continue
- wrap
- reflect
- clamp
- preserve tension
- later reverse those continuations through inverse continuation

## Mental Model

This folder is about continuation laws.

It answers questions like:

- what happens when a value advances again
- what boundary behavior means
- what it means to reverse a repeated process
- how repetition preserves, transforms, or tensions structure

## Why It Matters

This folder is one of the main bridges between:

- arithmetic-like behavior
- traversal and carrier behavior
- inverse continuation and branching

Many things that look unrelated at the surface become repetition questions here.

## Scope

Keep this folder focused on continuation law and result structure. Higher-level symbolic authoring of `pow(...)`, `inverse(...)`, or `continue(...)` belongs in the symbolic layer.
