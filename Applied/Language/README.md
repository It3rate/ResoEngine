# Applied Language

This folder is for editable language-facing experiments built on top of the more abstract symbolic schemas.

The intended split is:
- `Core2.Symbolics/Conceptual`
  reusable cross-domain relation families
- `Applied/Language`
  concrete lexical inventories, notes, and experiments for human languages

This layer is allowed to be language-specific.
It should still try to map words onto structurally honest Core 2-style schemas rather than arbitrary symbolic tags.

## Current Direction

The first pass starts with:
- a draft English conceptual lexicon
- image-schema notes
- overlap between prepositions, temporal words, path verbs, scalar language, hierarchy, and kinship-like distance terms
- loose English encoding examples with approximate Core 2-style formulas

The point is not to finalize the lexicon yet.
The point is to see whether one small symbolic family can be reused across:
- space
- time
- motion
- scale
- social distance and hierarchy
- procedural and narrative flow

## Surface Categories We Want To Compare

The current working categories are:
- prepositions and spatial relators
- temporal words
- motion and path verbs
- dimensional and scalar adjectives
- kinship, intimacy, and hierarchy language
- image-schema notes and localist-style families

These categories overlap heavily.
For example:
- `before` and `after` are both temporal words and directional offsets
- `approach` is a motion verb built on source-goal plus adjacency
- `upper management` reuses vertical hierarchy
- `close friend` reuses proximity in social space

That overlap is exactly what this folder is meant to expose.
