# Core 2 Interpretation Resolution

This folder contains interpreted resolution and layered quantity reading.

It takes the primitive support and resolution ideas from `Core2/Resolution` and turns them into explicit reading machinery: ladders, frames, quantization, representative readings, and residual detail.

## What Lives Here

This area handles things like:

- layered quantities
- resolution ladders
- resolution frames
- quantization rules and quantizers
- representative and residual readings

## Mental Model

If `Core2/Resolution` defines the primitive support laws, this folder shows what those laws look like in an interpreted measured setting.

This is where support becomes an actual reading practice rather than only an abstract law distinction.
