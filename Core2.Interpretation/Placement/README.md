# Core 2 Interpretation Placement

This folder contains placement and embedding helpers for interpreted Core 2 structure.

Placement here is not just coordinate assignment. It is the act of situating Core 2 structure in a frame or host context while preserving the meaning of the original element.

## What Lives Here

This area handles things like:

- positioned axes
- placement extensions
- embedding a local structural reading into a larger interpreted frame

## Mental Model

Use this folder when a native element already exists, but it now needs a place, position, or embedding in an interpreted situation.

This is where local structure becomes situated structure.
