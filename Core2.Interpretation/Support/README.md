# Core 2 Interpretation Support

This folder contains shared interpretation-side support structures.

These are not meant to be generic utility classes detached from Core 2. They are shared interpreted structures that help several interpretation domains work together while staying in Core 2 terms.

## What Lives Here

This area currently handles things like:

- interpreted containment
- grouped interpreted elements
- shared interpretation-side operation catalogs
- support structures for preserved interpretation tension

## Mental Model

Use this folder when a concept is clearly about interpreted structure, but does not belong only to analysis, placement, traversal, or units.

This is the interpretation-side support substrate, not a generic helpers bucket.
